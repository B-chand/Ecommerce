# Ecommerce React App

A simple and responsive ecommerce frontend built with React. It includes product listings, category pages, a shopping cart, and reusable UI components.

## Features

- Responsive design
- Product listing and product details
- Category-based navigation
- Shopping cart UI
- Footer with social icons
- React Router navigation
- Tailwind CSS styling

## Tech Stack

- React
- React Router DOM
- Tailwind CSS
- DaisyUI
- React Toastify
